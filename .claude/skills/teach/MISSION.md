# Mission: AI 全栈工程师

## Why

独立从 0 到 1 开发并上线一个有 AI 功能的完整产品——前端界面、后端 API、AI 集成全部自己掌控，不依赖别人写后端。

## Success looks like

**主线：Node.js 全栈 AI 应用（第 1–6 周）**
- 能独立写一个 Node.js + TypeScript 后端，对外暴露 REST API
- 能安全地调用 LLM API（Claude / OpenAI），实现对话、内容生成等功能
- 能用数据库持久化用户数据和对话历史
- 能把应用部署到互联网，给别人一个 URL 就能访问
- 遇到全栈问题时，知道在哪一层找根因

**副线：Java Spring Boot（第 6 周起）**
- 能读懂并修改公司的 Java 后端代码
- 能用 Spring Boot 重新实现同一个 AI 应用，理解 Java 生态
- 熟悉 Maven、Spring MVC、JPA 等企业级常用组件

## Constraints

- 每天 1-2 小时学习时间
- 前端（Vue 3 + React）已熟练，TypeScript 有实际项目经验
- 后端完全从零开始
- 偏好项目驱动式学习
- 公司使用 Java 后端（增加了学习 Java 的必要性）

## Out of scope

- 机器学习 / 模型训练（用 API，不训练模型）
- 深入 DevOps / Kubernetes（能部署到 PaaS 即可）
- 移动端开发
- Java 和 Node.js 同时双线学习（先 Node.js 跑通，再切 Java）
