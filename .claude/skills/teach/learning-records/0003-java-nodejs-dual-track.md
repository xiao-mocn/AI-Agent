# 用户确认双阶段后端学习计划

用户希望同时掌握 Node.js 和 Java 后端——动机是公司使用 Java，且 Java 相对通用。
经讨论，确认采用串行而非并行策略：先用 Node.js 跑通完整 AI 应用（主线），
第 6 周起用 Java Spring Boot 重写同一应用（副线）。
用户同意这个方向。

**Implications**:
- 前 6 周教学内容不变，聚焦 Node.js + TypeScript
- 第 6 周起引入 Java，届时可以复用已有的 HTTP/数据库/AI 概念，只需关注语言差异
- 遇到公司 Java 代码时，可以先用"读懂"的目标来降低难度
