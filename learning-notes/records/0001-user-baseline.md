# 用户起点：前端熟练，后端从零

用户同时掌握 Vue 3 和 React，有 TypeScript 生产项目经验（hmi-front 项目），
对后端、数据库、LLM API 完全没有接触。
这意味着教学可以跳过前端基础，直接从后端 HTTP 概念入手，
且 TypeScript 不是障碍——后端代码风格上可以直接用 TS，降低认知负荷。

**Evidence**: 用户自述，并有 hmi-front 项目（Vue 3 + TS + Pinia + Axios）作为佐证。

**Implications**: 
- 第 2 课开始就可以直接写 TypeScript 后端代码，不需要先讲 JS 基础
- HTTP 请求/响应的概念用户肯定懂（Axios），可以快速类比
- 数据库和 SQL 需要从零讲起
- LLM API 需要从零讲起
